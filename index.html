// commands

Every command
you need.

All slash-based. No prefix, no confusion.

/rust dropsPost current Twitch Drop campaigns
/rust kick-dropsPost current Kick Drop campaigns
/rust skinsPost latest Steam Workshop skin releases
/rust dlcPost available Rust DLC with prices
/rust devblogsPost last 3 Facepunch devblogs
/rust commitsPost latest Facepunch GitHub commits
/rust oxidePost latest Oxide/uMod updates
/rust bansPost latest org bans
/rust bancheck [steamid]Check Steam VAC, Game and Community bans
/rust playtime [steamid]Show Rust hours + total Steam playtime
/rust wipeShow next forced wipe date
/rust faqFrequently asked Rust questions
/rust setregion [cc]Set store region for skin/DLC prices
/rust filter [on/off]Enable or disable content filter — admin only
/wire setupchannelsCreate or link feed channels — admin only
/wire repopulateImmediately post latest content — admin only
/wire setregion [cc]Set the store region for this server
/wire pingCheck if the bot is online
/raid calculateCalculate explosives needed to raid any structure
/raid compareCompare raid cost between two structures
/recycle [item]Show what you get from recycling an item
/craft [item]Show crafting recipe and materials needed
/blueprint [item]Check if an item has a blueprint and its cost
/cctv [monument]Look up CCTV camera codes for any monument
/puzzle [monument]Show puzzle/loot room guide for a monument
/stats [steamid]Show detailed Rust player statistics
/wipestatsShow wipe statistics for your server
/eventtimersShow timers for in-game events
/lfg postPost a looking-for-group listing
/lfg searchFind players to team up with
/lfg profileView your LFG profile and stats
/leveling rankCheck your XP rank on this server
/leveling leaderboardView the top players by XP
/leaderboardShow the server-wide activity leaderboard
/rustplus setupConnect your Rust+ server to Discord
/rustplus mapShow the live map of your connected Rust server
/rustplus infoShow server info (population, map size, queue)
/rustplus timeShow current in-game time
/rustplus teamShow your team members and their online status
/rustplus monumentsList active monuments and their status
/rustplus eventsShow active in-game events
/rustplus chatSend a message to in-game chat from Discord
/rustplus statusCheck Rust+ connection status
