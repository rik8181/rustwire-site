// what it does

Everything Rust,
in one bot.

Set it up once, forget about it. RustWire watches the Rust ecosystem 24/7 and posts directly to your channels.

🎮
Twitch & Kick Drops
Automatically posts active drop campaigns the moment they go live. Never miss a free skin again.
🎨
Skins & DLC
New Steam Workshop skins and DLC releases posted instantly with prices in your server's region.
📰
Devblogs & News
Facepunch devblogs, community posts and general Rust news delivered the moment they're published.
🛠️
Commits & Oxide
Facepunch GitHub commits and Oxide/uMod plugin updates tracked and posted automatically.
🚫
Ban Logs
Latest BattleMetrics org bans, plus your own server ban log with full user profile information.
⚙️
Player Tools
Player stats, Steam VAC checks, hours verification with tier roles, raid calculator and more.
🔒
Content Filter
Automatic detection and removal of CSAM spam links with instant ban. Opt-in per server.
🧹
Wipe Schedule
Track forced wipe dates, set custom wipe countdowns, and notify your community automatically.
🔍
LFG System
Looking-for-group board with team size, playstyle filters and automatic expiry.
// commands

Every command
you need.

All commands are slash-based. No prefix, no confusion.

/rust dropsPost current Twitch Drop campaigns
/rust kick-dropsPost current Kick Drop campaigns
/rust skinsPost latest Steam Workshop skin releases
/rust dlcPost available Rust DLC with prices
/rust devblogsPost last 3 Facepunch devblogs
/rust commitsPost latest Facepunch GitHub commits
/rust oxidePost latest Oxide/uMod updates
/rust bansPost latest org bans (BattleMetrics API required)
/rust bancheck [steamid]Check Steam VAC, Game and Community bans
/rust playtime [steamid]Show Rust hours + total Steam + top 5 games
/rust wipeShow next forced wipe date
/rust filter [on/off]Enable or disable content filter on this server (admin)
/rust setregion [cc]Set store region for skin/DLC prices (e.g. GB, US)
/wire setupchannelsCreate feed channels in this server (admin)
/wire repopulateImmediately post latest content to all feed channels (admin)
/wire setregionSet region for this server
/lfg postPost a looking-for-group listing
/lfg searchFind players to team up with
/raidCalculate raid costs for any structure
/wipeschedule setupConfigure wipe countdown for your server
/wipeschedule enable/disableToggle wipe notifications

What's new.

Recent updates and improvements to RustWire.

Jun 2026
New
Kick Drops Support
RustWire now tracks and posts Rust Kick Drop campaigns automatically via kick.facepunch.com — same as Twitch Drops, with a dedicated #kick-drops channel.
Jun 2026
New
Ban Log Channel
Manual and automatic bans now post a detailed embed with user profile, account age, join date, ban reason and who banned them.
Jun 2026
New
Per-Server Content Filter
Server admins can now enable or disable the content filter independently on their server using /rust filter on/off.
Jun 2026
Fix
Autopost Performance
Shared feeds (commits, oxide, drops) are now fetched once per cycle instead of once per server, eliminating timeout spam on large deployments.
May 2026
Improve
10,000 Users Milestone
RustWire reached 10,000 Discord users across 100+ servers. Privileged intent review submitted to Discord.
