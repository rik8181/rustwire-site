/rust dropsPost current Twitch Drop campaigns
/rust kick-dropsPost current Kick Drop campaigns
/rust skinsPost latest Steam Workshop skin releases (region-aware)
/rust dlcPost available Rust DLC with prices (region-aware)
/rust devblogsPost last 3 Facepunch devblogs
/rust communityPost last 3 Facepunch community posts
/rust generalPost last 3 general Rust news posts
/rust commitsPost latest Facepunch GitHub commits
/rust oxidePost latest Oxide/uMod updates
/rust bansPost latest org bans (BattleMetrics API required)
/rust bancheck [steamid]Check Steam VAC, Game and Community bans
/rust playtime [steamid]Show Rust hours + total Steam playtime + top 5 games
/rust inventory [steamid]View a player's public Steam inventory
/rust wipeShow next forced wipe date
/rust faqFrequently asked Rust questions
/rust roadmapShow the current Rust development roadmap
/rust setregion [cc]Set store region for skin/DLC prices (e.g. GB, US)
/rust filter [on/off]Enable or disable content filter — admin only
/wire setupchannelsCreate or link feed channels on this server — admin only
/wire repopulateImmediately post latest content to all feed channels — admin only
/wire setregion [cc]Set the store region for this server (e.g. GB, US)
/wire pingCheck if the bot is online and responsive
/wire posthoursPost the Steam hours verification card — admin only
/raid calculateCalculate explosives needed to raid any structure

What's new.

Recent updates and improvements to RustWire.

Jun 2026
New
Kick Drops Support
RustWire now tracks and posts Rust Kick Drop campaigns automatically — same as Twitch Drops, with a dedicated #kick-drops channel.
Jun 2026
New
Ban Log Channel
Manual and automatic bans now post a detailed embed with user profile, account age, join date, ban reason and who issued the ban.
Jun 2026
New
Per-Server Content Filter
Server admins can now enable or disable the content filter independently on their own server using /rust filter on/off.
Jun 2026
Fix
Autopost Performance
Shared feeds (commits, oxide, drops) are now fetched once per cycle instead of once per server, eliminating timeout spam on large deployments.
May 2026
Milestone
10,000 Users
RustWire reached 10,000 Discord users across 100+ servers. Privileged intent review submitted to Discord.
